# WBDV WINTER 2021 - CPNT 262 Assignment 1: classList Toggle

Creating Hamburger menu and Content drawer functionalities using Javascript classList toggle.

**Author and Repo maintainer:** Swapna Sama

**GH repo :** [https://github.com/swappnasama/cpnt262-a1.git](https://github.com/swappnasama/cpnt262-a1.git)

**GH Pages site :** [https://swappnasama.github.io/cpnt262-a1/](https://swappnasama.github.io/cpnt262-a1/)

**Discription :** 

Beautiful Homes is interior designing and home staging company. They want to build a e-marketing website to expand their customer range. Website describes all the information about company's contact information, Employee, Gallery and testimonials.

Hamburger menu is built based on javascript classList and Covered Industries section is built like a content drawer/ accordian.
